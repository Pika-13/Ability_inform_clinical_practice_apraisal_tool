# Ability to Inform clinical practice
*This repository contains project of metaresearch study, which is a systematic review of surgical RCT**

## Aims
### Assess shortcoming in pretrial planning, measured by report of 2 elements: STUDY DESIGN & ENDPOINT
### Assess clinical meaningfulness incorporation in pretial planning:
#### 1) proportion of RCTs aiming for a MCID for the effect size
#### 2) proportion of RCTs interpretating results according to clinical relevance
